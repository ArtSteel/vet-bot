# VetAdvice AI (ВетСоветник)

> Open Source Telegram-бот и AI-ассистент для владельцев домашних животных.

Репозиторий: https://github.com/ArtSteel/vet-bot

## Почему появился этот проект

Проект родился из личной ситуации. Когда моя собака Несси заболела, на руках были сложные результаты анализов, а ближайший прием у ветеринара был только на следующий день.  
Нужно было быстро понять, насколько это срочно.

Так появился VetAdvice AI:
- быстро расшифровывать анализы по фото/PDF,
- объяснять результаты простым языком,
- подсвечивать риски по системе "Светофор":
  - красный = критично,
  - желтый = нужно наблюдать,
  - зеленый = вероятно норма.

## Возможности

- AI-консультации по симптомам, уходу и питанию
- OCR/vision-анализ фото и PDF документов
- Медкарта питомца (мульти-питомец + контекст)
- Монетизация из коробки (YooKassa), промокоды, рефералка
- Админ-панель (статистика, финансы, рассылки, промокоды)
- Middleware-логирование действий пользователей и тарифа

## Технологический стек

- **Framework:** Aiogram 3.x
- **База данных:** PostgreSQL (SQLAlchemy Async)
- **State management:** Redis (FSM storage)
- **Контейнеризация:** Docker + Docker Compose
- **AI-интеграция:** OpenAI-compatible API (смена провайдера через base URL)

## Быстрый запуск (Docker)

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ArtSteel/vet-bot.git
cd vet-bot
```

2. Создайте `.env`:
```bash
cp .env.example .env
```

3. Заполните обязательные переменные в `.env`:
- `TELEGRAM_BOT_TOKEN`
- `AI_API_KEY`
- `REDIS_URL`
- параметры PostgreSQL и YooKassa (если используются)

4. Запустите проект:
```bash
docker-compose up -d --build
```

## Примечания по конфигурации

- AI-клиент работает через OpenAI-compatible интерфейс; провайдер меняется через `AI_BASE_URL`.
- При наличии `POSTGRES_*` используется PostgreSQL (приоритетно).
- Redis используется как хранилище состояний FSM.

## Медицинский дисклеймер (строго)

**VetAdvice AI не является медицинской услугой и не заменяет очный прием у лицензированного ветеринарного врача.**  
Ответы ИИ могут содержать ошибки, неполные данные или небезопасные рекомендации.  
При любых острых, тяжелых или быстро ухудшающихся симптомах немедленно обращайтесь в ветеринарную клинику.

Используя проект, вы принимаете ответственность за интерпретацию и применение полученной информации.
